2. Sequence: GOLD to NMDC ETL Pull

sequenceDiagram
  participant GOLD as GOLD REST API
  participant Client as GoldApiClient
  participant Trans as GoldStudyTranslator
  participant Mongo as MongoDB (nmdc)
  participant Stitch as Records Stitching
  participant Mint as ID Minting

  Note over GOLD, Mint: Dagster Job: gold_study_to_database

  Client->>GOLD: GET /biosamples?study_id=X
  GOLD-->>Client: biosamples[]

  Client->>GOLD: GET /projects?study_id=X
  GOLD-->>Client: projects[]

  Client->>GOLD: GET /analysis_projects?study_id=X
  GOLD-->>Client: analysis_projects[]

  Client->>Trans: raw GOLD data
  Trans->>Trans: Map fields via gold-to-mixs.sssom
  Trans->>Trans: Filter by sequencingStrategy
(Metagenome, Metatranscriptome)
  Trans->>Mongo: Store nmdc:Biosample,
nmdc:NucleotideSequencing,
nmdc:Study

  Note over Mongo, Mint: Dagster Job: database_records_stitching

  Stitch->>Mongo: Fetch existing biosamples
  Stitch->>GOLD: Fetch GOLD projects for
gold_biosample_identifiers
  Stitch->>Mint: mint_id("nmdc:NucleotideSequencing", N)
  Mint-->>Stitch: new IDs
  Stitch->>Mongo: Create missing
data_generation_set records
  Stitch->>Mongo: Update INSDC biosample
identifiers from GOLD

3. Sequence: Portal Submission to NMDC to NCBI

sequenceDiagram
  participant PM as Project Manager
  participant Portal as NMDC Submission Portal
  participant PgDB as PostgreSQL (JSONB)
  participant PTrans as SubmissionPortalTranslator
  participant Mongo as MongoDB (nmdc)
  participant NCBI as NCBI

  PM->>Portal: Fill StudyForm
(GOLDStudyId, JGIStudyId,
NCBIBioProjectId)
  PM->>Portal: Fill MultiOmicsForm
(jgi_samp_id, jgi_sample_name, ...)
  Portal->>PgDB: Store submission (JSONB)
  Portal->>Portal: Validate (Pydantic schemas)

  Note over Portal, PgDB: Status: InProgress then
SubmittedPendingReview then
Approved

  PgDB->>PTrans: Approved submission
  PTrans->>PTrans: Translate to NMDC schema
  PTrans->>Mongo: submit_metadata_to_db()

  Note over Mongo, NCBI: Dagster Job: nmdc_study_to_ncbi_submission_export

  Mongo->>Mongo: Fetch biosamples by study_id
  Mongo->>Mongo: Fetch NucleotideSequencing records
  Mongo->>Mongo: Fetch DataObjects and
LibraryPreparation
  Mongo->>NCBI: NCBISubmissionXML
(BioSample + SRA + GenBank)
